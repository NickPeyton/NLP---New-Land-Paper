4132 - Thame, Oxfordshire in the Seventeenth Century



Dataset Type
============
Relational database



Dataset Format
==============

text - data only (30 tab delimited text files  + 729 RTF files)



Data Notes
==========

found in some of the tables. Synonyms of all these terms are included in
the "CodesStatus" table:

"beggard"
"p"

"junior"
"the late"
"esq"

"man"



Using the Data
==============
Users should note that the field name for surnames is "sname" in all
tables except the "CodesNames" table, where the field name is "surname".

Users who have received the Access 97 version should install the both
database files in: "c:\thamedata\" to ensure that the front end for the
database will be able to locate the data.

The text version of the dataset contains tab delimited text files (one
per database table) and a set of Rich Text Format (RTF) files. There is
no front end for the text version of the dataset. Users should note that
the documentation refers to the front end, but this is only available in
the Access 97 version of the dataset.

The RTF files in the text version of the dataset hold the contents of
the Microsoft Word documents that are embedded in the original database
as the "script" field in the "probatescript" table. In the tab delimited
text file version of this table (probatescript.tab) the "script" field
contains the name of the related RTF file that now holds the
information.

Newer versions of Access may not display the full content of the database, 
users should use the rtf and tab files instead.
